# HyperClusterAI

Universal Spatial-Spectral Intelligent Segmentation System.

## Features

- RGB and hyperspectral image support
- KMeans clustering
- PCA + KMeans clustering
- Spatial-spectral clustering
- Autoencoder clustering
- Deep Embedded Clustering (DEC)
- Intelligent recommendation engine
- Metrics dashboard
- Confusion matrix visualization
- Method comparison dashboard
- Export system
- Runtime and memory performance dashboard

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Technologies Used

- Python
- Streamlit
- TensorFlow
- Scikit-learn
- NumPy
- Matplotlib
- Pandas

## Deployment

HyperClusterAI is ready for deployment on Streamlit Community Cloud or Hugging Face Spaces.

For Streamlit Community Cloud:

1. Push this project to a GitHub repository.
2. Connect the repository in Streamlit Community Cloud.
3. Select `app.py` as the entry point.
4. Deploy.
